# Log Analyzer

Анализатор логов веб-сервера с графическим интерфейсом

## Особенности
- Парсинг логов в формате Apache/Nginx
- Хранение данных в PostgreSQL
- Визуализация статистики
- Экспорт в CSV, Parquet и Excel

## Требования
- Python 3.8+
- PostgreSQL
- PySide6
- psycopg2
- pandas
- matplotlib
